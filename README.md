# aws-signing-for-zap

aws-signing-for-zap is an Help Add On Script for Zed Attack Proxy. It signs all requests for AWS.

### Prerequisites

Jython 2.5.4-rc

ZAP Proxy 2.7.0

### Installing

For Jython installation:

```
http://www.jython.org/downloads.html
```
